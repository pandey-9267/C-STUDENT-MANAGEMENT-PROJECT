# 🎓 Student Result Management System

A simple console-based **Student Result Management System** developed in **C**. The program allows users to manage student records, calculate results, search students, and save records to a text file.

## ✨ Features

- Add Student Records
- Calculate Total & Average Marks
- Automatic Grade Calculation
- Display All Students
- Search Student by Roll Number
- Save Records to `students.txt`

## 🛠️ Technologies Used

- C Programming
- GCC Compiler
- Visual Studio Code

## 📁 Project Structure

```
Student-Result-Management-System/
│── PROJECT.C
│── students.txt
└── README.md
```

## ▶️ How to Run

```bash
gcc PROJECT.C -o project
```

gcc PROJECT.C -o project
.\project.exe
